Detailed Description

The Blockfield class represents a blockfield in the gameworld.

Blockfield has BasicProduction:

  • Money = 2
  • Food = 0
  • Wood = 0
  • Stone = 5
  • Ore = 2

Building in the blockfield takes time. So buildings get extra hold-marker.

Tile supports 3 buildings.
