Detailed Description

The OreDeposit class represents an ore deposit in the gameworld.

OreDeposit has BasicProduction:

  • Money = 3
  • Food = 0
  • Wood = 0
  • Stone = 2
  • Ore = 5

Building in the ore deposit takes time. So buildings get extra hold-marker.

Tile supports 2 buildings.
