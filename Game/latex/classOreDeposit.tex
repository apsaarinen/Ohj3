\subsection{Detailed Description}
The \hyperlink{classOreDeposit}{Ore\-Deposit} class represents an ore deposit in the gameworld. 

\hyperlink{classOreDeposit}{Ore\-Deposit} has Basic\-Production\-: \par

\begin{DoxyItemize}
\item Money = 3
\item Food = 0
\item Wood = 0
\item Stone = 2
\item Ore = 5
\end{DoxyItemize}

Building in the ore deposit takes time. So buildings get extra hold-\/marker.

Tile supports 2 buildings.
