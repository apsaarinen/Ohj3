\subsection{Detailed Description}
The \hyperlink{classBlockfield}{Blockfield} class represents a blockfield in the gameworld. 

\hyperlink{classBlockfield}{Blockfield} has Basic\-Production\-: \par

\begin{DoxyItemize}
\item Money = 2
\item Food = 0
\item Wood = 0
\item Stone = 5
\item Ore = 2
\end{DoxyItemize}

Building in the blockfield takes time. So buildings get extra hold-\/marker.

Tile supports 3 buildings.
